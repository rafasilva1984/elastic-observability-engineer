# Escopo do curso oficial — Create Visualizations

Fonte: Learning Plan oficial (learn.elastic.co): aprofundar os fundamentos
do Lens criando múltiplos tipos de visualização — barras verticais e
horizontais, heat maps e outros — escolhendo o formato certo para cada
pergunta.

Mapeamento (fontes oficiais):
- Lens (tipos suportados, Suggestions, color mapping, sampling):
  elastic.co/docs/explore-analyze/visualize/lens
- Bar charts (vertical/horizontal, stacked/grouped, breakdown, legendas):
  elastic.co/docs/explore-analyze/visualize/charts/bar-charts
- Heat maps (horizontal/vertical axis, cell value, custom color ranges):
  elastic.co/docs/explore-analyze/visualize/charts/heat-map-charts
- Metric charts (primary/secondary metric, tendência, dynamic coloring,
  progress bar): elastic.co/docs/explore-analyze/visualize/charts/metric-charts
Nota: interatividade (controls, drilldowns) é o assunto do Vídeo 12.
