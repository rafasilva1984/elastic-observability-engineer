## 📊 Web Logs — Visão Operacional

**Pergunta que este painel responde:** o tráfego web está saudável agora?

**Como ler:**
- O número grande é o **volume total de bytes** servidos no período.
- A série temporal mostra o **fluxo de requisições** — quedas bruscas ou picos merecem atenção.
- As barras mostram os **países de origem** do tráfego.
