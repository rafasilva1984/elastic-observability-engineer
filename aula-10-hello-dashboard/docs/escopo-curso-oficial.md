# Escopo do curso oficial — Hello Dashboard

Fonte: Learning Plan oficial (learn.elastic.co): criar seu primeiro
dashboard do Kibana do zero — usar o editor Lens para construir gráficos,
adicionar texto descritivo com Markdown e organizar painéis num layout
polido: o fluxo essencial para construir qualquer dashboard.

Mapeamento (fontes oficiais):
- Create a dashboard from the UI: fluxo de criação, modo de edição,
  opções de exibição (margins, panel titles, sync cursor/tooltips,
  store time), título/descrição/tags
  (elastic.co/docs/explore-analyze/dashboards/create-dashboard)
- Text panels: painéis Markdown (GitHub-flavored) e reuso via Visualize
  Library (elastic.co/docs/explore-analyze/visualize/text-panels)
- Lens: arrastar-e-soltar, Suggestions, edição direto no dashboard
  (elastic.co/docs/explore-analyze/visualize/lens)
- Sample data (web logs) para praticar sem ingestão
  (elastic.co/docs/explore-analyze/kibana-data-exploration-learning-tutorial)
Nota: aprofundamento de tipos de visualização fica no Vídeo 11; controls e
drilldowns no Vídeo 12 (cursos oficiais dedicados).
