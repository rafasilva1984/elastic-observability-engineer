# Escopo do curso oficial — Introduction to Kibana

Fonte: Elastic Observability Engineer — Self-Paced Learning Plan (learn.elastic.co),
curso "Introduction to Kibana".

Descrição oficial do curso (resumida em nossas palavras): aprender a interface
central do Kibana e suas capacidades através de exploração hands-on de sample
datasets — navegar pela UI, interagir com dashboards e visualizações, e ter um
primeiro contato com recursos avançados como Maps, Canvas e Machine Learning —
construindo a base necessária para todos os módulos seguintes da certificação.

O que esta aula cobre, ancorado na documentação oficial:

- **Discover**: explorar campos e dados, filtrar com KQL, ajustar time range,
  salvar sessões.
  (https://www.elastic.co/docs/explore-analyze/discover/discover-get-started)
- **Lens**: editor de visualização por arrastar-e-soltar, sugestões
  automáticas de gráfico, tipos de visualização.
  (https://www.elastic.co/docs/explore-analyze/visualize/lens)
- **Dashboards**: combinar visualizações num painel único e interativo.
  (https://www.elastic.co/docs/explore-analyze)
- **Sample data sets**: eCommerce, Flights, Web Logs — para praticar sem
  ingestão.
  (https://www.elastic.co/docs/explore-analyze/kibana-data-exploration-learning-tutorial)

Recursos avançados (Maps, Canvas, Machine Learning) são apenas apresentados
em nível de preview nesta aula, conforme a descrição do curso oficial — o
aprofundamento acontece em módulos específicos mais adiante.
