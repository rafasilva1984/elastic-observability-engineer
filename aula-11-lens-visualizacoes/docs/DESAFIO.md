## 🔒 Desafio autônomo — "Create Custom Maps Visualization"
*(mesmo nome do challenge oficial EOE 6.2)*

**Missão A (Maps):** o sample de web logs tem coordenadas
(`geo.coordinates`). Sem passo a passo, crie uma visualização de **Maps**
mostrando a origem geográfica das requisições e adicione ao catálogo.
Pesquise na doc oficial de Maps como criar a camada.

**Missão B (critério):** para cada pergunta abaixo, escolha o gráfico,
construa e justifique em 1 frase:
1. "O tráfego de bytes está subindo ou caindo esta semana?"
2. "Quais os 5 sistemas operacionais mais comuns (machine.os.keyword)?"
3. "Qual a proporção de cada código HTTP no total?"

### Critérios de aceite
- [ ] Mapa renderizando pontos/clusters reais
- [ ] 3 gráficos com a justificativa pergunta→formato
- [ ] Nenhum anti-padrão da aula cometido

### Validação
Dashboard [ONP] Catálogo Lens com 10 painéis (6 + mapa + 3).

### Dica (só se travar)
Maps fica no menu de visualizações; a camada usa a data view e o campo geo.
